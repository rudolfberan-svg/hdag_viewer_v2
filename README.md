This application is an interactive, web-based 3D molecular structure viewer tailored for analyzing the cryo-EM structure of the Hepatitis Delta Antigen (HDAg) oligomerization domain. This app incorporates data from Itskanov et al. 2026 PNAS doi: 10.1073/pnas.2519809123 and Moraleda et al. 2000 JVI doi: 10.1128/jvi.74.12.5509-5515.2000. Developed to aid in the visual exploration of viral protein and RNA interactions, the tool provides an intuitive interface for mapping critical structural and functional regions.  Key Features: Interactive 3D Visualization: Seamlessly rotate, zoom, and inspect the cryo-EM structure of HDAg.  Interface Mapping: Effortlessly visualize viral protein and RNA chains, highlighting the secondary structures that drive viral assembly.  Oligomerization Analysis: Easily map and identify mutationally critical residues involved in key dimer and multimer interfaces.  Sleek UI: Features a modern dark-mode interface optimized for high-contrast scientific analysis.
